Tabletalk-Restaurant Management System

Tabletalk is a full-stack restaurant management platform built using the MERN stack. The system streamlines restaurant operations including menu management, order processing, secure authentication, role-based access control, and payment integration. It is designed with scalability, security, and performance optimization in mind.

Tech Stack

  • React.js
  • Node.js
  • Express.js
  • MongoDB
  • JWT Authentication
  • Nodemailer (Email Service)
  • Stripe Payment Integration
  • Firebase Hosting
  • Vercel Deployment
Live Demo GitHub Repository

Features

  • Secure user authentication with JWT & HTTP-only cookies
  • Role-based dashboards (Admin & User)
  • Dynamic menu management (CRUD operations)
  • Online food ordering and cart system
  • Stripe payment gateway integration
  • Order tracking and status management
  • Search, filter, sort, and pagination functionality
  • Fully responsive UI for desktop, tablet, and mobile devices

Challenges & Solutions

Implementing secure authentication and handling protected API routes was a major challenge. This was solved using JWT tokens stored in HTTP-only cookies and middleware-based role verification. Additionally, efficient pagination and server-side filtering were implemented to optimize performance for large datasets.

Future Improvements

Planned improvements include real-time order updates using WebSockets, advanced analytics dashboards for restaurant owners, and automated email notifications for order confirmations.
