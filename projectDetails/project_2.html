ZenTask - Task Management Platform

ZenTask is a full-stack task management and micro-job marketplace platform where buyers can post tasks, workers can complete tasks for rewards, and admins manage the entire ecosystem. The system is built with scalability, role-based access control, and real-world workflow automation in mind.

Tech Stack

  • React (Vite)
  • Node.js
  • Express.js
  • MongoDB
  • Firebase Authentication
  • JWT & Protected Routes
  • Image Upload (Cloud)
Live Demo GitHub Repository

Features

  • Role-based dashboards (Admin, Buyer, Worker)
  • Secure authentication using Firebase & JWT
  • Task creation, submission, approval, and rejection system
  • Protected routes with PrivateRoute implementation
  • Axios interceptors for secure API communication
  • Responsive UI with reusable components
  • Admin controls for users, tasks, and platform analytics

Challenges & Solutions

Implementing role-based access control was one of the main challenges. This was solved by combining Firebase authentication with JWT tokens and protected routes, ensuring users only access authorized dashboards. Axios interceptors were used to handle secure API calls efficiently.

Future Improvements

Planned enhancements include real-time notifications, wallet & withdrawal systems, advanced admin analytics, task rating systems, and performance optimization using caching and pagination.
