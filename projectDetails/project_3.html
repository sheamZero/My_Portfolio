Craft!qo - Art & Craft Store Website

A comprehensive art and craft e-commerce platform that connects creative artists with customers worldwide. Built with modern web technologies, Craft!qo offers a seamless shopping experience with diverse art and craft subcategories, secure user authentication via Firebase, and a user-friendly interface powered by React and Tailwind CSS. The platform enables artists to showcase their work while providing customers with easy browsing, detailed product information, and efficient order management.

Tech Stack

  • React
  • Node.js
  • Express.js
  • MongoDB
  • Firebase
  • Tailwind CSS
Live Demo GitHub Code

Features

  • User authentication and login with Firebase
  • Browse art and craft items across multiple categories and subcategories
  • Detailed product pages with descriptions and images
  • Add new items to the store (for authenticated users)
  • Update and manage your own items
  • View all items and filter by category
  • Responsive design optimized for all devices
  • Beautiful animations with Lottie animations
  • Smooth image slider carousel (Swiper)
  • Community engagement features

Challenges & Solutions

Managing real-time inventory updates across multiple categories was solved through efficient state management with React Context. Implementing smooth animations while maintaining performance was achieved using Lottie for lightweight animations. Authentication security was handled by leveraging Firebase's robust security features and private routes for protected pages.

Future Improvements

Planned enhancements include adding a shopping cart and checkout system, implementing payment gateway integration, adding user reviews and ratings, creating an admin dashboard for analytics, implementing search and advanced filtering options, and adding wishlists functionality for personalized shopping experiences.
